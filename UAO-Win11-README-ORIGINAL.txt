UAO 2.50－Windows 11 安裝工具
================================

需要的檔案
----------
只需要「UAO-Win11-Installer.exe」。
字碼表與離線安裝腳本都已內嵌在 EXE，不需要另外複製 CMD 或 NLS。

安裝方式
--------
1. 請先儲存並關閉正在進行的工作。
2. 執行 UAO-Win11-Installer.exe。
3. Windows 詢問是否允許變更裝置時，選擇「是」。
4. 按「1. 準備安裝檔與備份」。
5. 顯示準備完成後，按「2. 重開到修復選單」。
6. 電腦重新啟動後，依序選擇：
   疑難排解 → 進階選項 → 命令提示字元
7. 在黑色命令提示字元視窗輸入：

   C:\u

8. 如果顯示找不到檔案，改輸入：

   D:\u

9. 畫面顯示「SUCCESS - UAO 2.50 installed.」代表安裝成功。
10. 按任意鍵後輸入 exit，選擇繼續進入 Windows。
11. 開啟 ERP，測試原本無法輸入的字，例如「堃」。

解除安裝／還原
-------------
1. 再次執行 UAO-Win11-Installer.exe。
2. 按「解除 UAO／準備還原原始字碼表」。
3. 按「2. 重開到修復選單」。
4. 依序選擇：疑難排解 → 進階選項 → 命令提示字元。
5. 輸入 C:\r；若找不到檔案，改輸入 D:\r。
6. 顯示「SUCCESS - original Windows CP950 files restored.」代表還原成功。
7. 按任意鍵後輸入 exit，選擇繼續進入 Windows。

失敗保護
--------
安裝時如果複製或驗證失敗，工具會立即嘗試還原安裝前的微軟原始字碼表。
若顯示「CRITICAL - automatic restore failed」，請不要重新啟動，並拍下畫面尋求協助。

注意事項
--------
• 如果修復環境要求 BitLocker 復原金鑰，必須先輸入金鑰解鎖系統碟。
• 如果畫面顯示 ERROR，請先拍下錯誤畫面，不要重複執行安裝。
• Windows 大型版本更新可能還原 CP950 系統字碼表；若缺字再次出現，可重新安裝。
• 工具不會停用 Windows Defender。

檔案與備份位置
--------------
C:\u.cmd                         離線安裝指令
C:\r.cmd                         離線還原指令
C:\UAO-Win11\C_950.UAO.NLS      UAO 2.50 字碼表
C:\UAO-Win11\Backup-日期時間    安裝前的原始字碼表備份
